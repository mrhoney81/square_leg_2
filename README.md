# Nuke the UK - Operation Square Leg 2

My friends and I were discussing who's house we should decamp to in the event of nuclear war. Since I couldn't find a relatively up-to-date list of likely nuclear targets, I thought I'd create one in the spirit of the 1980s [Square Leg exercise](https://en.wikipedia.org/wiki/Square_Leg). 

So here you go - a modernised, machine-readable update to the 1980 Operation Square Leg nuclear target set for the UK, with an interactive warhead budget planner. You can export the results and load into [Alex Wellerstein's](https://doomsdaymachines.net/) excellent [NUKEMAP](https://nuclearsecrecy.com/nukemap/) to look at likely fallout plumes and fun stuff like that.

In case it's not obvious, all of the descriptions were generated by AI and locations are public domain knowledge. Yes the code is horrific but I didn't write it, Claude did.

---

## What is this?

**Operation Square Leg** (1980) was the last publicly available UK nuclear target assessment — a Home Office exercise modelling a 131-warhead Soviet strike, plus a companion MoD "Soviet hit list" of 106 targets. Both were declassified under the 30-year rule and are now ~45 years out of date.

This project creates an updated 2026 target set of **193 targets across 16 categories**, with:

- **Priority ranking** (1-193) for warhead-budget-constrained modelling
- **Tier system** (T1/T2/T3) from surgical counterforce to all-out
- **Yield and burst assumptions** based on plausible Russian delivery systems
- **Interactive web planner** with physics-based population avoidance filter
- **Export to NUKEMAP, KML, CSV, GeoJSON** for damage modelling

### What's changed since 1980?

| Then | Now |
|------|-----|
| 14 Regional Seats of Government | RSGs abolished. PINDAR, Northwood, Corsham (decommissioned). |
| 23 RAF bases | ~8 principal operating bases. Dozens closed. |
| Polaris at Faslane/Rosyth | Trident entirely at Faslane/Coulport. **Barrow builds every hull** (not on 1980 list). |
| No Rolls-Royce Raynesway | **Sole UK site for naval reactor cores.** Single point of failure. |
| No data centres | **Slough** — densest DC cluster in Europe. |
| No gas terminals | **St Fergus** — 40% of UK gas through one site. |
| No undersea cable targets | **Widemouth Bay** — primary transatlantic fibre bundle. |

## Interactive Warhead Budget Planner

Set a warhead budget and see the prioritised target list on an interactive map. Includes:

- Preset scenarios from surgical counterforce (9 warheads) to all-out (193)
- **Population avoidance filter** — uses Glasstone & Dolan overpressure scaling to exclude targets whose blast radius overlaps cities above a configurable population threshold
- Download **NUKEMAP CSV**, **KML** (Google Earth), or **full CSV** for the selected target set
- Mobile-friendly card layout

**[Try it locally](web/index.html)** or deploy to GitHub Pages.

## Tier system

| Tier | Count | Scenario |
|------|-------|----------|
| **T1** | 9 | **Nuclear counterforce only.** Faslane, Coulport, AWE, Devonport, Barrow, Raynesway, Lakenheath, Fylingdales. All rural — no city hit. |
| **T2** | 91 | **Full military + infrastructure.** Adds C2, intelligence, RAF/RN/Army bases, defence industry, energy, ports, data centres, telecoms. Striking London C2 nodes (PINDAR, MI5/MI6) is the escalation threshold. |
| **T3** | 93 | **All-out + counter-value.** All 39 cities (London 3 GZs + 36 others), 8 airports, remaining infrastructure. |

## Warhead budget scenarios

Russia has ~1,500 deployed strategic warheads. Most are allocated to the US. Realistic UK allocation: **30-80**.

| Budget | What it covers |
|--------|----------------|
| 9 | Nuclear forces only. No cities. |
| 20 | + C2 + GCHQ + Sellafield + St Fergus + Widemouth Bay. London C2 struck. |
| 30 | + Carrier base, QRA bases, strike/ISTAR, airlift, US bases. |
| 50 | Full military + key defence industry. Realistic ceiling for UK allocation. |
| 100 | Complete military + all energy + ports + data centres. |
| 193 | Everything including all cities and airports. |

## Yield and burst assumptions

| Target class | Yield | Burst | Rationale |
|---|---|---|---|
| Hardened military (BMEWS, bunkers, munitions depots) | 475 kt | Ground | Overpressure on hardened targets; accept fallout. |
| Soft strategic (AWE, GCHQ, bases, industry) | 100 kt | Air (500m) | Functional kill; limit fallout. |
| Naval bases (Faslane, Devonport, Portsmouth) | 475 kt | Air (500m) | Destroy ships/submarines in port. |
| Warhead storage (Coulport) | 475 kt | Ground | Hardened bunkers. |
| London | 800 kt x3 | Air (2000m) | Three offset GZs. |
| Other cities | 300 kt | Air (2000m) | Maximise blast + thermal radius. |
| Nuclear sites | 100 kt | Ground | Contamination is the weapon. |
| Energy / data centres / ports / airports | 100 kt | Air (500m) | Soft targets. |

## Sources

Sources include UK MoD estate publications, RAF/RN/Army official sites, BAE Systems/Rolls-Royce/Babcock annual reports, GCHQ/NSA public records, ONS population estimates, National Grid ESO network diagrams, TeleGeography Submarine Cable Map, and the original National Archives Square Leg documentation.

## Caveats

1. **Not intelligence.** Unclassified public-information assessment. Plausible doctrinal logic, not Russian operational plans.
2. **Coordinates are public-record precision.** Derived from published addresses, OSM, Wikipedia. Not operationally precise.
3. **Yield/burst are illustrative defaults.** Vary them for sensitivity analysis.
4. **Casualty modelling is out of scope.** This provides the target set; damage modelling is a separate step (NUKEMAP, HPAC, etc.).
